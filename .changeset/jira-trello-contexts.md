Add Jira and Trello as Contexts sources alongside Linear, built on a shared issue-provider abstraction.

- Connect Jira with a site URL, email, and API token to browse, search, and open issues from the Contexts sidebar, scope the feed to assigned issues or chosen projects, and append them straight into the composer.
- Connect Trello with an API key and token to browse, search, and open cards, scoped to your own cards or chosen boards.
- Linear, Jira, and Trello now share one unified inbox feed, so connect, search, detail preview, and append-to-composer work identically across every provider.
