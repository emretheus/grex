Improve the new-workspace start screen:

- The repository picker now filters as you type, shows the full `owner/repo` name alongside the folder name, and lets you press 1–9 to quickly pick one of the first nine repositories.
- A new Alt+R shortcut opens the repository picker, and Alt+P now opens the model picker on the start page (previously it only worked inside a conversation).
- Your model, reasoning effort, plan mode, and fast mode picks now survive switching to another workspace and back instead of resetting to defaults.
